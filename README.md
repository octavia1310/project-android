List To List – Smart To-Do List Application

List To List adalah aplikasi To-Do List berbasis Android yang dirancang untuk membantu pengguna dalam mencatat, mengatur, dan memantau tugas harian secara lebih terstruktur.
Aplikasi ini dikembangkan dari referensi aplikasi GitHub sederhana, kemudian ditingkatkan dengan berbagai fitur modern seperti login, kalender, prioritas tugas, riwayat, hingga notifikasi pengingat.

Dengan tampilan UI yang lebih interaktif dan user-friendly, List To List membantu pengguna tetap produktif dan tidak melewatkan tugas penting setiap hari.

Fitur Utama
1. Login & Registrasi
Pengguna dapat membuat akun baru.
Autentikasi aman menggunakan email & password.
Tersedia fitur Forgot Password melalui email.
2. Manajemen Tugas (CRUD)
Tambah tugas baru (judul, deskripsi, tanggal, waktu).
Edit tugas yang sudah dibuat.
Hapus tugas.
Tandai tugas sebagai Selesai atau Batal.
3. Kategori & Prioritas
Kategori tugas (Work, Personal, dll).
Prioritas tugas: High, Medium, Low.
4. Kalender
Menampilkan tugas berdasarkan tanggal.
Pengguna dapat melihat jadwal harian dengan lebih rapi.
5. Riwayat Tugas
Menampilkan tugas yang sudah selesai atau dibatalkan.
Tugas dapat dikembalikan lagi ke status aktif.
6. Notifikasi Pengingat
Pengguna mendapatkan pengingat otomatis sebelum deadline.
7. Profil Pengguna
Edit profil (nama).
Upload foto profil.
Informasi tersimpan di database.

Teknologi yang Digunakan
1. Java — Bahasa utama untuk logika aplikasi Android.
2. XML — Untuk desain tampilan (UI).
3. Android Studio — IDE pengembangan aplikasi.
4. MySQL + PHP API — Backend untuk penyimpanan data (users, tasks, riwayat, reset password).
5. Volley — Komunikasi antara aplikasi dan API server.
6. Notification Manager — Mengelola notifikasi pengingat.

Cara Menjalankan Aplikasi
1. Clone repository dari GitHub.
2. Buka project menggunakan Android Studio.
3. Pastikan backend (API PHP + MySQL) sudah dijalankan.
4. Atur file konfigurasi URL API sesuai server.
5. Build & Run aplikasi pada emulator atau perangkat Android.
6. Aplikasi siap digunakan untuk mengelola aktivitas harian!
